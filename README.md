# lcdapi

## Synopsis

C++ API for lcdproc clients, original project at http://www.c-sait.net/lcdapi/

Now at https://github.com/spdawson/lcdapi

## Getting started

Assuming that the latest version is `X.Y`:

```sh
tar xaf lcdapi-X.Y.tar.gz
cd lcdapi-X.Y
libtoolize --force
autoreconf -i
./configure
make
sudo make install
```
